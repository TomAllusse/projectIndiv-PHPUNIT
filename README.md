# projectIndiv-PHPUNIT

# Installation
<br>
<summary> Installation de composer </summary>
<br>

```bash
composer init
```

<summary> Installation de PHPUNIT </summary>
<br>

```bash
composer require --dev phpunit/phpunit
```
