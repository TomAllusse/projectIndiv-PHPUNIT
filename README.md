# projectIndiv-PHPUNIT

# Installation

<summary style="margin: 10px auto;"> Installation de composer </summary>

```bash
composer init
```

<summary style="margin: 10px auto;"> Installation de PHPUNIT </summary>

```bash
composer require --dev phpunit/phpunit
```
